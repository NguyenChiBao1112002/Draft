*Lesson 3: Introduce Selenium---------------------------------------------------------------------
- Selenium là một library (quan trọng cần học là WebDriver(1 trong 4 thành phần của Selenium))

-Phân biệt: Framework và Library
	+ Library: cung cấp các tiện ích, chức năng có sẵn, làm nhiệm vụ nào đó
	+ Framework: gồm nhiều library, là một khung chương trình sẵn có, 
	khi code phải tuân thủ một số nguyên tắc cấu trúc của Framework
